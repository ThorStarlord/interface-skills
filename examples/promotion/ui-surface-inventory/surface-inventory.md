# Surface Inventory: Multi-Layer Dashboard

## Surface 1: Global Navigation
- **Type**: Sidebar
- **Density**: Compact
- **Visual Vocabulary**: Material Design
- **Interactive Elements**:
  - Home link
  - Settings gear
  - User profile avatar

## Surface 2: Data Grid
- **Type**: Content Area / Table
- **Complexity**: High
- **Ambiguous Boundary**: Shared scroll with the header.
- **Interactive Elements**:
  - Column headers (sortable)
  - Row checkboxes
  - Action menu for each row

## Surface 3: Contextual Filter Bar
- **Type**: Tool Bar
- **State**: Persistent
- **Interactive Elements**:
  - Search input
  - Date range picker
  - Filter chips

## Surface 4: Notification Toast
- **Type**: Overlay
- **State**: Transient
- **Interactive Elements**:
  - Close button
  - 'Undo' link
