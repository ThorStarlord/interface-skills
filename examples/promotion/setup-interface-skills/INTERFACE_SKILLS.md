# Interface Skills Repository Policy

This file acts as the single source of truth for the **Interface Skills UI Specification Layer** within this repository. All AI coding agents must consult this policy and any active spec packages before editing or auditing UI routes and components.

## Specs Directory Root
All active UI specifications reside in:
- `docs/saas-frontend/specs/`

## Policy Directives
1. **Specs are the Source of Truth:** Code must match specifications. Never update code without first checking or updating the spec package.
2. **Deterministic Handoffs:** Maintain semantic linkage and ID propagation across all phases (Brief, Calibration, Blueprint, etc.).
3. **No Unplanned Changes:** All spec-to-code translations must go through approved promotion plans.
