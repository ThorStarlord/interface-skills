# Example: Onboarding Flow

> **Status: Coming soon.**

This spec package will demonstrate the full Interface Skills workflow applied to a multi-step onboarding wizard — covering welcome screens, progress indicators, form steps, and success states.

## Planned deliverables

- `brief.md` — product brief targeting first-time user activation
- `visual-calibration.md` — warm, spacious, low-friction aesthetic calibration
- `flow.md` — user journey from signup confirmation to first value moment
- `blueprint.md` — wireframe spec for each wizard step
- `system.md` — design tokens optimised for trust and momentum
- `screen-spec.md` — screen contracts for each onboarding step
- `component-specs/` — anatomy specs for the StepIndicator, WelcomeHero, and FormStep components
- `microcopy.md` — approved copy for every step heading, CTA, and error message
- `acceptance.md` — testable acceptance criteria (including a11y and mobile breakpoints)
