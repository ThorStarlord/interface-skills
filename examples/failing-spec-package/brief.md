# Brief: Post Composer Widget

## 1. Goal

Allow a user to compose and submit a post from the dashboard.

## 2. Primary user

Marketing coordinator. Uses the dashboard daily. Moderate SaaS familiarity.

## 3. Primary action

Submit a post.

## 4. Secondary actions

- Attach an image.
- Preview the post before submitting.

## 5. Why it matters

Posts are the core output of the platform. Any friction here reduces daily active usage.

## 6. Success criteria

- A user can submit a post in under 45 seconds.
- The submit button shows a loading state during submission.

## 7. Constraints

- Use the existing brand colours.
- Must work on mobile and desktop.
