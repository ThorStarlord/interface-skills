# Vague Language Translator

When a user uses vague language, prompt them to clarify using this translation matrix.

| Vague Word   | Translation Options |
|--------------|---------------------|
| Clean        | More whitespace, fewer borders, lower color variety, simpler hierarchy |
| Modern       | Flat surface style, large radius, muted palette, contemporary type, generous spacing |
| Professional | Restrained color, dense information, predictable layout, low ornament |
| Playful      | Rounded shapes, warmer palette, expressive empty states, softer motion |
| Premium      | High contrast, restrained palette, elegant typography, fewer visible controls |

| Pop          | High saturation primary colors, heavy typography, subtle bounce animations, distinct outlines |
| Minimalist   | Maximum whitespace, monochrome palette, hidden controls until hover, absolute grid alignment |
| Corporate    | Standardized blue/gray palette, dense tabular data, low-radius buttons, high contrast text |
| Edgy         | Dark mode default, sharp corners (0px radius), brutalist borders, neon accents, mono fonts |
| Fun          | Organic shapes, pastel or bright palette, spring physics motion, emoji or illustrative icons |
