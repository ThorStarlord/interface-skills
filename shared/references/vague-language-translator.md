# Vague Language Translator

When a user uses vague language, prompt them to clarify using this translation matrix.

| Vague Word   | Translation Options |
|--------------|---------------------|
| Clean        | More whitespace, fewer borders, lower color variety, simpler hierarchy |
| Modern       | Flat surfaces, large radius, muted palette, contemporary type, generous spacing |
| Professional | Restrained color, dense information, predictable layout, low ornament |
| Playful      | Rounded shapes, warmer palette, expressive empty states, softer motion |
| Premium      | High contrast, restrained palette, elegant typography, fewer visible controls |
