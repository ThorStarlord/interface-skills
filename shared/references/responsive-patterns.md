# Responsive Patterns

This document outlines standard responsive behaviors for layout components.

## Breakpoints
- Mobile (sm): < 640px
- Tablet (md): 640px - 1024px
- Desktop (lg): 1024px - 1280px
- Widescreen (xl): > 1280px

## Common Transitions
- **Sidebar**: Hidden behind hamburger menu on mobile, visible on desktop.
- **Grids**: 1 column on mobile, 2 on tablet, 3+ on desktop.
- **Modals**: Full-screen on mobile, centered dialog on desktop.
- **Navigation**: Bottom tab bar on mobile, top or side bar on desktop.
