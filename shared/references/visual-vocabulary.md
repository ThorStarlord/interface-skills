# Visual Vocabulary

This document establishes the canonical vocabulary for describing visual UI decisions across all skills.

## Layout Archetypes
- Sidebar App
- Centered Card
- Split Panel
- Dashboard Grid
- Wizard

## Density
- Sparse
- Medium
- Dense

## Shape Language
- Sharp (0-2px radius)
- Mildly Rounded (4-8px radius)
- Pill-like (Fully rounded)

## Surface Style
- Flat
- Outlined
- Elevated (Shadows)
- Glassy
- Card-heavy
